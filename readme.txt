
#Falling Block Console Game

Overview:

This is a simple console-based falling block game written in C for Windows. The player controls a cursor that moves left and right to avoid falling obstacles. The game features a score system, lives, and gradually increasing speed.

Features:

Player cursor is blue for easy visibility.

Move the player using:

A / D keys

Left / Right arrow keys

Avoid falling obstacles.

Score increases as obstacles pass.

Lives decrease on collision; game ends when lives reach 0.

Game speed increases as the score grows.

Game Controls
Key	Action
A / Left Arrow	Move player left
D / Right Arrow	Move player right
How to Run

Make sure you have a C compiler installed (like Dev-C++ or Visual Studio).

Copy the game.c file into your project directory.

Compile the program:

gcc game.c -o game.exe
